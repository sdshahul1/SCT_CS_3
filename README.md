# Password Strength Checker

A Python tool that analyzes and rates the strength 
of a password based on multiple security criteria.

## 📌 Task
This project was completed as Task 3 of my Cyber 
Security Internship at **SkillCraft Technology**.

## 🔑 About Password Strength Checker
Weak passwords are one of the biggest security 
vulnerabilities. This tool helps users understand 
how strong their password is and gives suggestions 
to improve it.

## ⚙️ Features
- Checks password length
- Detects uppercase letters (A-Z)
- Detects lowercase letters (a-z)
- Detects numbers (0-9)
- Detects special characters (!@#$%)
- Gives a score out of 6
- Rates password as WEAK, MODERATE or STRONG
- Provides improvement suggestions

## 🛠️ Technologies Used
- Python
- Regular Expressions (re module)
